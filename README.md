# Julián Matter — Personal Website

Personal portfolio website for Julián Matter — Civil Engineer specialized in Data Science & AI applied to infrastructure and construction.

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure & semantic markup |
| CSS3 | Styling, layout, animations |
| Vanilla JavaScript | Interactivity, modal system, i18n |
| Google Fonts (DM Sans + DM Serif Display) | Typography |

No frameworks, no dependencies, no build step — just clean and fast static files.

---

## ✨ Features

- **Bilingual (ES / EN)** — full i18n system with language persistence via `localStorage`
- **Project modal** — detailed project view with image gallery and lightbox
- **Smooth scroll** — native scroll with fixed nav offset
- **Responsive** — mobile-first layout, hamburger menu on small screens
- **SEO ready** — structured data (JSON-LD), Open Graph, Twitter Card meta tags
- **Accessible** — ARIA labels, keyboard navigation support

## 🚀 Running Locally

No build step needed. Just clone and open:

```bash
git clone https://github.com/julianmatter/julianmatter.github.io.git
cd julianmatter.github.io
# Open index.html in your browser, or use a local server:
npx serve .
```
